Método Rota Descrição
GET /user/:name Rota para verificação de usuário por nome.
GET /users Rota para exibição de usuários.
POST /users Rota para adicionar usuário.
DELETE /users/:name Rota para deletar usuários do sistema por nome.
PUT /users/:id Rota para editar os usuários do sistema por ID.
GET /users/access/:name Rota para contagem de leituras do usuário por nome.
